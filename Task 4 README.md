# 3D Mechanical Assembly – Flange Coupling

## Overview

This mini project involves designing and assembling a complete flange coupling using Autodesk AutoCAD. Individual components including the flange, shaft, bolt, nut, and rectangular pin were modeled separately and assembled into a functional mechanical system. Engineering drawings and rendered views were also prepared.

---

## Objectives

- Design individual mechanical components.
- Assemble components into a complete flange coupling.
- Apply professional CAD modeling techniques.
- Generate engineering drawings and rendered views.
