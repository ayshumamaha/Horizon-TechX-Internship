# Mechanical Flange Drawing using AutoCAD

## Overview

This project involves the preparation of a detailed two-dimensional engineering drawing of a circular flange using Autodesk AutoCAD. The drawing includes front, sectional, and isometric views with complete dimensions, centerlines, annotations, and standard mechanical drafting conventions.

---

## Objectives

- Prepare a detailed flange drawing.
- Apply orthographic projection techniques.
- Create sectional and isometric views.
- Produce manufacturing-ready engineering documentation.
