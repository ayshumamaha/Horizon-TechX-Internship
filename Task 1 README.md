# Residential Floor Plan Design using AutoCAD

## Overview

This project focuses on the development of a detailed two-dimensional residential floor plan using Autodesk AutoCAD. The drawing represents the second floor of a residential building and includes three bedrooms, a kitchen, a lobby, a staircase, and a combined toilet and bath. Standard architectural drafting practices were followed to ensure dimensional accuracy, readability, and efficient space utilization.

---

## Objectives

- Create a complete residential floor plan using AutoCAD.
- Apply architectural drafting standards.
- Develop a functional and well-organized room layout.
- Produce a fully dimensioned engineering drawing.
