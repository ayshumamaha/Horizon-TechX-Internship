# 3D Mechanical Assembly (Axial Piston Pump)

## Overview

This project focuses on creating a three-dimensional mechanical assembly using Autodesk AutoCAD. The assembly consists of multiple components assembled together to form an axial piston pump. Sectional views, orthographic drawings, rendering, and material assignment were used to improve visualization and engineering documentation.

---

## Objectives

- Develop a complete 3D mechanical assembly.
- Apply solid modeling techniques.
- Generate sectional and orthographic views.
- Produce realistic rendered visualizations.
